## Asennus

Asenna riippuvuudet komennolla:

```bash
poetry install
```

## Komennot

### Ohjelma

Komennot suoritetaan virtuaaliympäristössä komennolla:

```bash
poetry shell
```

Ohjelma suoritetaan komennolla:

```bash
python3 src/main.py
```

### Testaus

Testit suoritetaan komennolla:

```bash
pytest src
```

### Pylint

Pylint raportti saadaan komennolla:

```bash
pylint src
```
